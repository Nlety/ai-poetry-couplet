## 项目简介

AI 对联诗词是一款传统文化创作工具，可以生成对联、藏头诗、七言诗、五言诗和宋词。

## 核心功能

- 🎊 **春联对联**：新年婚庆等场合对联
- 🎯 **藏头诗**：自定义藏头字的诗歌
- 📝 **七言/五言诗**：经典诗歌创作
- 🎶 **宋词**：按词牌填词
- 💾 **收藏作品**：保存喜欢的作品

## 本地运行

直接用浏览器打开 `index.html` 即可使用。
